# Formulário de Inscrição - Nascer Renascer

Este é um projeto de formulário de inscrição desenvolvido para o cliente Nascer Renascer. O objetivo deste projeto é criar um formulário de inscrição online para o evento "Nascer Renascer", onde os participantes podem se inscrever para participar.

## Funcionalidades Principais

- Preenchimento fácil do formulário de inscrição.
- Feedback visual para indicar campos preenchidos incorretamente.
- Botão de envio que processa os dados do formulário e os envia para o site mailchimp.

## Tecnologias Utilizadas

- HTML: Estrutura do formulário e elementos de entrada.
- CSS: Estilos e layout do formulário.
- JavaScript: Validação de entrada e interatividade do formulário.

## Como Usar

1. Acesse o formulário de inscrição através do link: [Nascer Renascer - Formulário de Inscrição](https://rmaxyz.github.io/nascerrenascer.github.io/)
2. Preencha os campos do formulário de inscrição.
3. Certifique-se de preencher corretamente todos os campos obrigatórios.
4. Clique no botão "Enviar" para processar a inscrição.
